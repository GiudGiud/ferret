# QuasistaticFieldAux

!syntax description /AuxKernels/QuasistaticFieldAux

## Overview

Computes the electric field due to $\mathbf{E} = -K \nabla \Phi_\mathrm{E}$. An optional conversion factor $K$ can be used to change the units that are output.
